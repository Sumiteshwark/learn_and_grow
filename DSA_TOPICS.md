## 🔵 Two Pointers & Arrays

| Algorithm | Topic | Complexity | Key Idea |
|---|---|---|---|
| **Floyd's Tortoise & Hare** | Two Pointers / Linked List | O(n) time, O(1) space | Fast + slow pointer to detect cycle or find middle |
| **Dutch National Flag (3-way partition)** | Two Pointers / Sorting | O(n) time, O(1) space | Partition array into 3 groups in one pass |
| **Kadane's Algorithm** | Arrays / DP | O(n) time, O(1) space | Max subarray sum by tracking local vs global max |

---

## 🔍 Searching & Sorting

| Algorithm | Topic | Complexity | Key Idea |
|---|---|---|---|
| **Binary Search** | Binary Search | O(log n) | Halve search space each step on sorted data |
| **Binary Search on Answer** | Binary Search | O(n log n) | Guess answer, validate with a feasibility check |
| **Merge Sort** | Sorting / Divide & Conquer | O(n log n) | Split, sort halves, merge — stable sort |
| **Quick Sort** | Sorting | O(n log n) avg, O(n²) worst | Partition around pivot — in-place |
| **Counting Sort** | Sorting | O(n + k) | Sort by frequency when range k is small |
| **Topological Sort (DFS)** | Graphs / DAG | O(V + E) | Post-order DFS, push to stack on finish |
| **Kahn's Algorithm (BFS topo sort)** | Graphs / DAG | O(V + E) | Process nodes with in-degree 0 iteratively |

---

## 🌐 Graph Algorithms

| Algorithm | Topic | Complexity | Key Idea |
|---|---|---|---|
| **BFS** | Graphs / Trees | O(V + E) | Shortest path in unweighted graph, level-order |
| **DFS** | Graphs / Trees | O(V + E) | Cycle detection, connected components, paths |
| **Dijkstra's Algorithm** | Graphs (weighted) | O((V + E) log V) | Greedy shortest path — min-heap, non-negative weights only |
| **Bellman-Ford** | Graphs (weighted) | O(V × E) | Shortest path with negative weights, detects negative cycles |
| **Floyd-Warshall** | Graphs | O(V³) | All-pairs shortest path via DP |
| **Union-Find (DSU)** | Graphs | O(α(n)) ≈ O(1) | Track connected components, detect cycles — path compression + rank |
| **Kruskal's MST** | Graphs | O(E log E) | Sort edges, add smallest edge that doesn't form cycle (uses DSU) |
| **Prim's MST** | Graphs | O(E log V) | Grow MST greedily from a node using min-heap |
| **Tarjan's SCC** | Graphs (advanced) | O(V + E) | Strongly connected components using DFS + low-link values |
| **Bridges & Articulation Points** | Graphs (advanced) | O(V + E) | Find critical edges/nodes in undirected graph |

## 📝 String Algorithms

| Algorithm | Topic | Complexity | Key Idea |
|---|---|---|---|
| **KMP (Knuth-Morris-Pratt)** | Strings | O(n + m) | Build LPS (failure) array to skip re-comparison on mismatch |
| **Rabin-Karp (Rolling Hash)** | Strings | O(n + m) avg | Hash the pattern, slide hash window — good for multi-pattern search |
| **Z-Algorithm** | Strings | O(n + m) | Z-array stores length of longest prefix match at each position |
| **Manacher's Algorithm** | Strings | O(n) | Find all palindromic substrings in linear time |

## 🧮 Dynamic Programming Algorithms

| Algorithm | Topic | Complexity | Key Idea |
|---|---|---|---|
| **0/1 Knapsack** | DP | O(n × W) | Include or exclude each item |
| **Unbounded Knapsack** | DP | O(n × W) | Reuse items — scroll forward in same row |
| **LCS (Longest Common Subsequence)** | DP on Strings | O(n × m) | 2D DP table comparing characters |
| **LIS (Longest Increasing Subsequence)** | DP | O(n log n) with patience sort | Binary search + dp array for optimal |
| **Edit Distance** | DP on Strings | O(n × m) | Insert / delete / replace cost table |
| **Matrix Chain Multiplication** | DP (interval) | O(n³) | Optimal way to parenthesize matrix multiplications |

## 🔢 Math & Bit Algorithms

| Algorithm | Topic | Complexity | Key Idea |
|---|---|---|---|
| **Sieve of Eratosthenes** | Math | O(n log log n) | Find all primes up to n |
| **GCD (Euclidean Algorithm)** | Math | O(log min(a,b)) | gcd(a, b) = gcd(b, a % b) |
| **Fast Exponentiation (Binary Exponentiation)** | Math | O(log n) | Compute a^n by squaring — avoids n multiplications |
| **Brian Kernighan's Bit Count** | Bit Manipulation | O(set bits) | n & (n-1) clears lowest set bit — count iterations |
| **XOR Swap / Duplicate Finder** | Bit Manipulation | O(n) | XOR cancels pairs — finds single element |
